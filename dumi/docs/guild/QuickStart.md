# 快速开始

通过此教程，让我们一步一步开始使用吧。

## 一、安装 amiya

### 前置安装

Amiya 依赖于 [Ant Design V4](https://ant-design.gitee.io/index-cn) ，请预先安装好。

### 使用 npm 安装

```bash
npm i amiya -S
```

### 使用 yarn 安装

```bash
yarn add amiya -S
```

## 二、测试使用

```tsx
import React from 'react'

export default function Demo() {
  return '<AyButton>Amiya</AyButton>'
}
```
